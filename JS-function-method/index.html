apply

虽然在一个独立的函数调用中，根据是否是strict模式，this指向undefind或者window。但是还是可以控制this的指向的。

要制定函数的this指向哪个对象，可以用函数本身的apply()方法，它接受两个参数，第一个参数是this要绑定的对象，第二个参数是Array，是函数本身的参数列表。

还有一个类似的方法是call(),区别在于：

  • apply()把参数打包为一个Array再传入，call()把参数按顺序传入。
