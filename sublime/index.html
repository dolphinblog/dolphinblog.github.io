sublime text 3打开时总是弹出Package Control Messages(我遇到的是Anaconda插件升级信息), 如何关闭?

打开Preferrence -> Package Settings -> Package Control -> Settings-User, 增加
"auto_upgrade": false, 这使得所有安装的package都不再自动升级, 也就不会弹出升级信息. 如需升级, 可选择Preferrence -> Package Control -> Package Control: Upgrade Package.
