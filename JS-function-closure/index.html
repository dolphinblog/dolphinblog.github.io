闭包

需要注意的是，返回的函数并不会立即执行，直到调用时（在函数名后面加()）才会执行。

这可能造成一些问题：

在上面的for (var i=1; i<=3; i++) { arr.push(function () { return i * i; });}
中，每次循环，都有一个函数function () { return i * i; }被添加到arr数组中。看看调用count()时返回什么：

返回3个函数，与预期相符。3个函数的返回值从左到右应该依次为1 4 9。

验证一下：

竟然都是16！What happend?

results是一个数组，数组的元素是三个函数，三个函数都引用了变量i，我们认为三个函数引用的i的值分别是1 2 3，这样三个函数对应的返回值就是1 4 9，现在结果却依次是16 16 16，问题出在哪？

问题出在了三个函数都引用的循环变量i中，所谓引用变量也好，常量也好，引用的实际都是一小块内存，只是对于常量来说，这一小块内存中的值是不变的（一直为常量的值），对于变量来说，这一小块内存的值是可变的。

这样，当这样调用results[0]();的时候，其实和临时调用function () { return i*i; }是等价的，这时候i的值其实是count()内部for循环退出时的值，就是4，所以三个函数的返回值都是16(4*4)。

因此需要注意：返回函数不要引用任何循环变量，或者后续会发生变化的变量。
