字符串

JS的字符串就是用''或""括起来的字符表示。

转义字符

如果'本身也是一个字符，可用""括起来，比如"I'm alright."。
如果字符串内部既包含'又包含"可使用转义字符\表示，比如'I'\m \"OK\"'。
关于这一点参考Python。

转义字符\可以转义很多字符，比如\n表示换行，\t表示制表符，字符\本身也要转义，所以\\表示的字符就是\。
