JS每创建一个对象，都会给这个对象设置一个原型对象。

用obj.xxx访问一个对象的属性时，JS引擎现在当前对象自己身上寻找该属性，如果没有找到，就在其原型对象上找，如果还没有找到，就一直上溯到Object.prototype对象，依然没有找到，就返回undefined。
