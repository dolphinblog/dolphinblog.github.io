操作DOM

DOM是Document Object Model的缩写。

HTML文档是一棵DOM树。可通过JS操作DOM，改变HTML的结构。

操作一个DOM节点实际上只有这么几种操作：

  • 遍历。遍历该DOM节点下的子节点，以便进行相应操作。
  • 更新。更新该DOM节点的内容，即更新该DOM节点表示的HTML的内容。
  • 添加。在该节点下面新增一个子节点，相当于动态增加了一个HTML节点。
  • 删除。将该节点从DOM树中删除，相当于从HTML中删除。

操作一个DOM节点之前，需要先拿到这个节点。最常用的方法是document.getElementById() document.getElementByTagName()，以及css选择器document.getElementByClassName()。

由于ID在HTML文档种是唯一的，所以document.getElementByID()可以直接定位唯一的一个DOM节点。document.getElementByTagName()和document.getElementsByClassName()总是返回一组DOM节点。
