操作文件

HTML控件中唯一可以上传文件的控件是<input type="file".

注意: 可以上传的前提是: enctype="multipart/form-data" method="post",这样浏览器才能正确编码并以multipart/form-data格式发送表单的数据.

处于安全考虑,浏览器只允许点击<input type="file">来选择本地文件,用JavaScript对<input type='file'>的value属性赋值是没有效果的.并且当用户选择某个文件后,JavaScript也无法获得文件的真实路径.

#### File API

> 由于JavaScript对用户上传的文件操作非常有限，尤其是无法读取文件内容，使得很多需要操作文件的网页不得不用Flash这样的第三方插件来实现。

随着HTML5的普及,新增的File API允许JS读取文件内容.

HTML5的File API提供了File FileReader两个对象,可以获得文件信息并读取文件.
